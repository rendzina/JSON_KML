# JSON_KML

> JSON conversion to KML

## Context

iPhone App [Sensor Log](https://itunes.apple.com/gb/app/sensorlog/id388014573?mt=8) is able to log spatial coordinates in real time as the phone is moved. The data that is logged can then be saved out as a JSON file. This is a Python script that can convert the JSON file to a KML file suitable for use in Google Earth.

## Usage

```
   parse_json_route.py -i <inputfile>
```
